difference between OS firmware and Boot firmware? --- BOOT firmware we DO NOT update -- its embedded in the product --  ****we only update the OS firmware.***  in order to update the firmware you have to put it into boot mode and then you can update the OS firmware.

sometimes units only support UP TO certain OS versions for a certain version of BOOT firmware -- we cant change the BOOT code/firmware but good to be aware of these limitations.

-- Hubs / LEDs---

Processor LED:
Green slow blink -- normal
Green rapid blink -- IDing processor
Blue blink -- software updating
Red Solid --- error -- usually a network error

Link LED:
Green slow blink -- normal
Red blink --- error -- comms error -- usually miswired cable or power over the cable

POE LEDs:
P1 Green -- Power
Fault -- amber -- P1 or P2 power fault
Link / Act solid/flash -- normal
